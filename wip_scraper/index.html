bid scraper

This is a tool I use to pull anonymised bid data.

It creates a CSV with three columns: Employer Name, Successful Bid, and Bid Rank.

to use it, I drag this link onto my bookmarks bar:

Scan CSM

(1) load the OCI bid page and change the number of rows per page to any value other than ten.

(2) wait for the page to reload.

(3) hit the new "scan CSM" bookmark.

(4) wait for the processing message to disappear from the top of the page.

(5) wait a few more seconds, then hit the button again.

(6) click the "download CSV" link at the top of the page.

Boom. now I have a CSV with all my WIP information so I can stay organized.
